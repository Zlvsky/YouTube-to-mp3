### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Zlvsky/YouTube-to-mp3.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Compile TypeScript
   ```sh
   tsc
   ```
4. Create .env file 
   ```sh
   API_KEY=PASTEYOURAPIKEY
   API_HOST=PASTEYOURRAPIDAPIHOST
   PORT=PORT
   ```
5. Run app
   ```sh
   node index.js
   ```
   ```
